Эндпоинт Тип запроса Описание Параметры
/cars?BrandId=1 GET Получить список всех автомобилей
параметры запроса для фильтрации:
BrandId, ModelId
/cars/:carId GET Получить автомобиль по id
/cars POST Создать автомобиль Пример тела запроса (body): 
    {
        "BrandId": 1,
        "ModelId": 1,
        "img": "audi-a1-1.jpeg",
        "bigImg": "audi-a1-1-big.jpeg",
        "price": 10000,
        "description": "Описание"
    }
 
/brands GET Получить список брендов
/models/:brandId GET Получить список моделей по бренду
